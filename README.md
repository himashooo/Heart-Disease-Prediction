# ❤️ Heart Disease Prediction

A machine learning web application that predicts the likelihood of heart disease based on patient health information.

## 🚀 Live Demo

👉 https://heart-disease-prediction555.streamlit.app/

## 🧠 Model

- **Algorithm:** Logistic Regression
- **Type:** Binary Classification
- **Target:** Heart Disease / No Heart Disease

## 📊 Features

The model uses features such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression
- Slope

## 🛠️ Tech Stack

Python • Pandas • Scikit-learn • Joblib • Streamlit
